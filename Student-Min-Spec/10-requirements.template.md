# Requirements

## Functional Requirements

### FR1: User Dashboard
- Display upcoming appointments
- Show appointment status

### FR2: Appointment Booking
- Users can book appointments
- Required fields: name, email, phone, date, time, service
- Status defaults to "Pending"

### FR3: Appointment Management
- View appointments
- Cancel appointments
- Reschedule appointments

### FR4: Profile Management
- View personal details
- View appointment statistics

### FR5: Settings
- Edit profile
- Update contact details
- Change password

### FR6: FAQ
- View expandable questions and answers

### FR7: Help Center
- Display contact info and help guides

### FR8: Authentication
- Login and logout functionality

## Non-Functional Requirements

- Performance: Load time ≤ 3 seconds
- Security: Protected user data
- Availability: 24/7 access
- Usability: Simple UI
