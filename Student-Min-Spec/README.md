# Dental Clinic Appointment System

## Description
A web-based system that allows patients to book and manage dental appointments online.

## Features
- Appointment booking
- Appointment management
- User dashboard
- Profile settings
- FAQ and Help Center

## Technologies Used
- HTML, CSS, JavaScript
- Node.js (optional)
- Database (MySQL/MongoDB/PostgreSQL)

## How to Run
1. Install dependencies
2. Start server
3. Open in browser
